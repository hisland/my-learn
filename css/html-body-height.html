我直接 position:absolute;
如果body设置position:relative我就相对于body
此时我的%高度是相对于:root
我会随着页面滚动, 如果想固定住, 使用 position:fixed;
我和body都是normal flow的方式

对 :root html body 设置 overflow:hidden; 全局滚动条都会消失

全局滚动条的 卷去高度 通过 document.body.scrollTop 得到, 不是 document.documentElement.scrollTop

html body 都可以用 margin, 它们都是display:block

document.documentElement.clientHeight 可得到 页面显示区域的高度
