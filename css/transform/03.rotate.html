rotate(a), 等同于 matrix(cos(a), -sin(a), sin(a), cos(a), 0, 0) a是角度,示例: 90deg = 100grad = 0.25turn ≈ 1.5708rad, 即使是0单位也不可缺,必须写成0deg之类
rotate(45deg)
rotate(100grad)
默认中心点是正中心, 可通过transform-origin修改
transform-origin:left top;, rotate(45deg)
transform-origin:right bottom;, rotate(45deg)
rotate3d(x, y, z, a), 公式太复杂,参考MDN
rotate3d(1, 1, 1, 45deg)
rotateX(a) is a shorthand for rotate3d(1, 0, 0, a)
rotateX(45deg)
rotateY(a) is a shorthand for rotate3d(0, 1, 0, a)
rotateY(45deg)
rotateZ(a) is a shorthand for rotate3d(0, 0, 1, a)
rotateZ(45deg)
