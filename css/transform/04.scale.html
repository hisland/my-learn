scale(sx) or scale(sx, sy), 等同于 matrix(sx, 0, 0, sy, 0, 0) 只有sx时, sy=sx
scale(1.2)
scale(0.8)
scale(0.8, 1.2)
负值翻转效果
scale(-0.8)
默认中心点是正中心, 可通过transform-origin修改
transform-origin:left top;, scale(0.8)
transform-origin:right bottom;, scale(0.8)
scale3d(sx, sy, sz), 无参考公式
scale3d(0.8, 0.8, 0.8)
scaleX(sx) is a shorthand for scale(sx, 1) or for scale3d(sx, 1, 1)
scaleX(1.2)
scaleY(sy) is a shorthand for scale(1, sy) or for scale3d(1, sy, 1)
scaleY(1.2)
scaleZ(sz) is a shorthand for scale3d(1, 1, sz)
scaleZ(1.2)
