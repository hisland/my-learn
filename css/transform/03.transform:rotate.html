rotate(a) 相当于 rotateZ(a) 相当于 rotate3d(0, 0, 1, a) 相当于 matrix(cos(a), -sin(a), sin(a), cos(a), 0, 0) a是角度,示例: 90deg = 100grad = 0.25turn ≈ 1.5708rad, 即使是0单位也不可缺,必须写成0deg之类
rotate(45deg)
rotateZ(45deg)
rotate(100grad)
默认中心点是正中心, 可通过transform-origin修改
transform-origin:left top;, rotate(45deg)
transform-origin:right bottom;, rotate(45deg)
rotate3d(x, y, z, a),参考MDN x,y,z确定一个点, 从原点到此点画一条直线作为旋转轴, 此直线是向量, 方向指向自己, a为正时顺时针转, a为负时逆时针转
rotate3d(1, 1, 1, 45deg)
rotateX(a) is a shorthand for rotate3d(1, 0, 0, a), 此时旋转轴为x方向
rotateX(45deg)
rotateY(a) is a shorthand for rotate3d(0, 1, 0, a), 此时旋转轴为y方向
rotateY(45deg)
rotateZ(a) is a shorthand for rotate3d(0, 0, 1, a), 此时旋转轴为z方向
rotateZ(45deg)
