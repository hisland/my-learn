god { cc1: (3 + 2); // + cc2: (3+2); cc3: (3 - 2); // - cc4: (3-2); cc5: (3 * 2); // * cc6: (3*2); cc7: (4 / 2); // / cc8: (4/2); }

单位

@base: 5; god { cc1: (3px + 5); cc2: (3em + 5); cc4: (3em + 5px); cc6: (#000 + 1); cc7: (@base + 1); }

优先级

@base: 5; god { cc1: ((@base + 5) * 3px); }
