Description: Use the chrome.sessions API to query and restore tabs and windows from a browsing session.
Availability: Dev channel only.
Permissions: "sessions"

chrome.sessions reference

Types

Filter

properties of Filter

maxResults ( optional integer )

The maximum number of entries to be fetched in the requested list. Omit this parameter to fetch the maximum number of entries (MAX_SESSION_RESULTS).

Session

properties of Session

lastModified ( integer )

The time when the window or tab was closed or modified, represented in milliseconds since the epoch.

tab ( optional tabs.Tab )

The tabs.Tab, if this entry describes a tab. Either this or Session.window will be set.

window ( optional windows.Window )

The windows.Window, if this entry describes a window. Either this or Session.tab will be set.

Device

properties of Device

info ( string )

Represents all information about a foreign device.

sessions ( array of Session )

A list of open window sessions for the foreign device, sorted from most recently to least recently modified session.

Properties

MAX_SESSION_RESULTS

chrome.sessions.MAX_SESSION_RESULTS
MAX_SESSION_RESULTS ( 25 )
The maximum number of Session that will be included in a requested list.

Methods

getRecentlyClosed

chrome.sessions.getRecentlyClosed(Filter filter, function callback)

Gets the list of recently closed tabs and/or windows.

Parameters

filter ( optional Filter )

callback ( function )

The callback parameter should specify a function that looks like this: 

function(array of Session sessions) {...};

sessions ( array of Session )

The list of closed entries in reverse order that they were closed (the most recently closed tab or window will be at index 0).The entries may contain either tabs or windows.

getDevices

chrome.sessions.getDevices(Filter filter, function callback)

Retrieves all devices with synced sessions.

Parameters

filter ( optional Filter )

callback ( function )

The callback parameter should specify a function that looks like this: 

function(array of Device devices) {...};

devices ( array of Device )

The list of Device objects for each synced session, sorted in order from device with most recently modified session to device with least recently modified session. tabs.Tab objects are sorted by recency in the windows.Window of the Session objects.

restore

chrome.sessions.restore(string sessionId, function callback)

Reopens a windows.Window or tabs.Tab, with an optional callback to run when the entry has been restored.

Parameters

sessionId ( optional string )

The windows.Window.sessionId, or tabs.Tab.sessionId to restore.

callback ( optional function )

If you specify the callback parameter, it should specify a function that looks like this: 

function(Session restoredSession) {...};

restoredSession ( Session )

A Session containing the restored windows.Window or tabs.Tab object.
